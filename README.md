# Cursor Website Clone

A pixel-perfect recreation of the Cursor AI code editor landing page, built with HTML and CSS.

## Sections Recreated

### 1. Navigation Bar

- Sticky header with logo, navigation links, and action buttons
- Features: Features, Enterprise, Pricing, Resources links
- Sign in and Download buttons

### 2. Hero Section

- Large headline with value proposition
- Download CTA button
- Dual screenshot showcase with layered images
- Custom background gradient

### 3. Trusted By Section

- Social proof text
- Logo grid featuring 8 company logos (Midjourney, OpenAI, Shopify, Perplexity, Notion, Ramp, Replit, Adobe)

### 4. Feature Sections (3 sections)

- **Agent Feature**: "Agent turns ideas into code" with screenshot
- **Autocomplete Feature**: "Magically accurate autocomplete" with video demo
- **Ecosystem Feature**: "Everywhere software gets built" with dual overlapping screenshots

### 5. Testimonials Section

- Grid layout with 6 testimonial cards
- Features quotes from:
  - Diana Hu (Y Combinator)
  - shadcn (shadcn/ui creator)
  - Andrej Karpathy (Eureka Labs)
  - Patrick Collison (Stripe)
  - ThePrimeagen
  - Greg Brockman (OpenAI)
- Each card includes profile image, name, and role

### 6. Frontier Section

- Three-column grid showcasing:
  - Access to best models
  - Complete codebase understanding
  - Enterprise development features
- Each card includes screenshot/demo image

### 7. Changelog Section

- Four-column grid showing recent updates
- Version badges (2.4, 2.3)
- Dates and feature descriptions

### 8. Team Section

- Centered heading about the team
- "Join us" CTA button
- Full-width team photo

### 9. Highlights Section

- Recent blog posts/updates
- Three highlight cards with titles, descriptions, and metadata
- Category tags (Product, Research)

### 10. Call-to-Action Section

- Large "Try Cursor now" heading
- Download button

### 11. Footer

- Five-column layout:
  - Product links
  - Resources links
  - Company links
  - Legal links
  - Connect (social media)
- Bottom bar with:
  - Copyright notice
  - SOC 2 certification badge
  - Theme switcher (3 icons: system, light, dark)
  - Language selector

## Fonts Used

- **Primary Font**: `Arial, Helvetica, sans-serif`
- **Font Weights**:
  - 100 (Thin) - Used for version badges
  - 200 (Light) - Used for subheadings and secondary text
  - 300 (Light) - Used for changelog titles and highlights
  - 400 (Regular) - Used for main headings and body text
  - 500 (Medium) - Used for CTA buttons

## Technical Details

- **Layout**: Flexbox and CSS Grid
- **Max Width**: 1400px for main content sections
- **Border Radius**: Consistent use of rounded corners (4px-30px)
- **Responsive**: Flexible layouts with percentage-based widths
- **Sticky Navigation**: Position sticky for navbar
- **Video Support**: Autoplay, muted, looping video for feature demo
